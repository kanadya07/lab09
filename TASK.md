# Задание к лабораторной работе №8

## Тема

Изучение систем автоматизации развертывания и управления приложениями на
примере Docker.

## Основное задание

1. Создать публичный репозиторий `lab08` на GitHub.
2. Взять проект из лабораторной работы №7.
3. Создать `Dockerfile`.
4. Собрать Docker-образ.
5. Запустить контейнер и проверить запись данных в файл логов.
6. Добавить сборку Docker-образа в `.travis.yml`.
7. Оформить отчет.
