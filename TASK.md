# Задание к лабораторной работе №9

## Тема

Изучение сервисов хранения артефактов на примере GitHub Release.

## Основное задание

1. Создать публичный репозиторий `lab09` на GitHub.
2. Взять проект из лабораторной работы №8.
3. Настроить данные для создания пакета.
4. Собрать архив проекта через CPack.
5. Создать тег `v0.1.0.0`.
6. Создать GitHub Release.
7. Загрузить собранный архив в релиз.
8. Оформить отчет.
